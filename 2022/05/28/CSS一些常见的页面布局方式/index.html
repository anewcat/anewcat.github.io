CSS的一些页面布局方式

Created2022-05-28|Updated2022-05-28|CSS
|Post View:

flex弹性盒（弹性布局）

弹性盒由弹性容器和弹性子元素组成

默认情况下弹性子元素在弹性盒内一行显示，从左到右（可通过这只direction属性改变弹性子元素排列方式）

开启方式：

  • 为弹性容器设置display属性为flex或inline-flex

主轴：弹性元素的排列方向

辅轴：与主轴垂直方向

属性：

设置在父容器上的属性

flex-direction

该属性指定了属性指定了内部元素是如何在 flex 容器中布局的，定义了主轴的方向

其属性值如下：

  • row：横向从左到右排列，默认的排列方式，主轴（左–>右）
  • row-reverse：反转横向排列,主轴（右–>左）
  • column：纵向排列。主轴（上–>下）
  • column-reverse：反转纵向排列，从后往前排，最后一项排在最上面。主轴（下–>上）

flex-wrap：

  • nowrap：默认值 元素不会自动换行

  • wrap：元素沿着辅轴方向自动换行，第一行在上方。

  • wrap-reverse：元素沿着辅轴反方向换行。第一行在下方。

flex-flow：

flex-flow属性是flex-direction属性和flex-wrap属性的简写形式。默认值为row nowrap

justify-content

该属性指定弹性子元素在父元素主轴上的排列方式

属性值如下：

  • flex-start：元素沿着主轴起边排列

  • flex-end：元素沿着主轴终边排列

  • center：元素在主轴居中紧挨着排列，如果子元素的宽度之和大于父元素（剩余空间为负），则会在两个方向上同时溢出

  • space-around：元素平均分布在主轴上，两边留有一半的间隔空间。如果剩余空间为负或者只有一个元素，则该值等同于center。否则，元素沿该行分布，且彼此间隔相等（比如是20px），同时首尾两边和弹性容器之间留有一半的间隔（1/2*20px=10px）。

  • space-between：元素平均分布在主轴上。如果剩余空间为负或者只有一个元素，则该值等同于flex-start。否则，第一个元素沿主轴起边排列，而最后一个元素沿主轴终边排列，然后剩余的元素分布在该行上，相邻项目的间隔相等。

  • space-evenly：均匀排列每个元素,每个元素之间的间隔相等

align-items

该属性指定了弹性子元素在父元素辅轴上的排列方式

  • stretch：默认值 ，如果元素未设置高度或设为auto ，将充满整个父元素高度，若设置告诉该属性与flex-start相同。

  • flex-start：元素沿着辅轴起边对齐

  • flex-end：沿着辅轴终边对齐

  • center：元素在辅轴居中。（如果父元素的尺寸小于弹性盒子元素的尺寸，则会向两个方向溢出相同的长度）。

  • baseline：元素第一行文字的基线对齐。

align-content：

该属性定义了多根轴线的对其方式。（有几行就有几根轴线）如果项目只有一根轴线，该属性不起作用。

  • flex-start：与辅轴的起点对齐。
  • flex-end：与辅轴的终点对齐。
  • center：与辅轴的中点对齐。
  • space-between：与辅轴两端对齐，轴线之间的间隔平均分布。
  • space-around：每根轴线两侧的间隔都相等。所以，轴线之间的间隔比轴线与边框的间隔大一倍。
  • stretch（默认值）：轴线占满整个辅轴。
设置在子元素上的属性

order：属性定义项目的排列顺序。数值越小，排列越靠前，默认为0.

flex-grow：属性定义项目的放大比例，默认为0，即如果存在剩余空间，也不放大。如果所有项目的 flex-grow 属性都为1，则它们将等分剩余空间（如果有的话）。如果一个项目的 flex-grow 属性为2，其他项目都为1，则前者占据的剩余空间将比其他项多一倍。

flex-shrink：属性定义了项目的缩小比例，默认为1，即如果空间不足，该项目将缩小。

如果所有项目的 flex-shrink 属性都为1，当空间不足时，都将等比例缩小。如果一个项目的 flex-shrink 属性为0，其他项目都为1，则空间不足时，前者不缩小。

负值对该属性无效。

flex-basis：元素在主轴上的基础宽高，即分配多余空间之前，弹性子元素占据的空间。

flex：该属性是 flex-grow, flex-shrink 和 flex-basis 的简写，默认值为 0 1 auto 。后两个属性可选。

flex：n（非负数字）代表 flex-grow：n；flex-shrink：1；flex-basis：0%；
flex：n1，n2（两个非负数字）代表flex-grow：n1；flex-shrink：n2； flex-basis：0%；
flex：L（长度或百分比）代表 flex-grow：1；flex-shrink：1；flex-basis：L；
flex：n，L（一个非负数字、一个长度或百分比）代表flex-grow：n；flex-shrink：1；flex-basis：L;
flex：1即为flex-grow：1；flex-shrink：1；flex-basis：0%；经常用作自适应布局，内容 区会 自动放大占满剩余空间。

​ 该属性有两个快捷值：

​ auto（1 1 auto）

​ none（0 0 auto）

align-self： 该属性允许单个项目有与其他项目不一样的对齐方式，可覆盖 align-items属性。默认值为 auto，表示继承父元素的 align-items 属性，如果没有父元素，则等同于 stretch。

定位布局

通过定位可将元素摆放到页面任意位置，通过设置position开启定位

position可选值：

  • static：默认值 元素静止，没有开启定位
  • relative：相对定位 相对定位是元素相对于其在原来标准流中位置进行移动，通过left，right，bottom，top属性进行调整

注意点：

  1. 相对定位的元素是不脱离文档流的，也就是说他是区分行内元素/块级元素/行内块元素，我们也可以为开启相对定位元素设置宽高
  2. 在同一方向上只能设置一个属性，即left，right选择一个属性设置，设置了top就不能设置bottom
  • absolute：绝对定位 默认情况下，无论有位祖先元素，都是以body作为参考点。但是，当祖先元素中有定位流元素(使用了绝对/相对/固定定位)则选择最近的开启定位的祖先元素作为参考点。

注意点：

  1. 固定定位的元素是脱离文档流的
  2. 和绝对定位一样，不区分区分行内元素/块级元素/行内块元素
  • fixed：固定定位 设置了固定定位的元素是相对于视口定位的，也就是说其不会随着滚动条的滚动而滚动，他始终处于以一个视口的位置，通过left，right，bottom，top属性调整其位置

注意点：

  1. 固定定位的元素是脱离文档流的因此不区分行内元素、块元素、行内块元素
  • sticky：粘滞定位 此定位结合了相对定位和固定定位，元素定位表现为在跨越特定阈值前为相对定位，之后为固定定位。

注意点：

  1. 粘滞定位的元素是不脱离文档流的
  2. 设置 position: sticky; 同时给一个（left，right，bottom，top） 之一即可

使用条件：

  1. 父元素不能overflow:hidden或者overflow:auto属性。
  2. 必须指定top、bottom、left、right4个值之一，否则只会处于相对定位
  3. 父元素的高度不能低于sticky元素的高度
  4. sticky元素仅在其父元素内生效

z-index

  • 对开启定位的元素，通过z-index指定元素的层级，值越大层级越高 优先显示
  • 如果元素层级一样 则优先显示靠下的
  • 祖先元素的层级再高也不会盖住后代元素

浮动布局

元素脱离文档流，浮在页面上层，通过float设置元素浮动

属性值：

  • none 默认值 不浮动
  • left 向左浮动
  • right 向右浮动

浮动的特点：

  • 主要作用：让页面中的元素可以水平排列 通过浮动可以制作一些水平方向的布局
  • 浮动元素不会盖住文字 文字会自动环绕在浮动元素的周围
  • 元素设置浮动后会脱离文档流会发生变化

脱离文档流后的特点：

  • 块元素：
  1. 不再独占一行
  2. 宽度和高度都被内容撑开
  • 行内元素： 行内元素脱离文档流以后会变成块元素 特点和块元素一样

清除浮动

利用伪元素选择器，添加空div，并设置 clear:both
